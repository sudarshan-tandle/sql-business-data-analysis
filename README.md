# Contoso Database Analysis – Business Intelligence with SQL
## Project Overview

This project focuses on analyzing the Contoso Retail Database using SQL to extract meaningful business insights from structured retail data. The database simulates a real-world retail business environment containing information related to customers, products, sales transactions, stores, currency exchange rates, and calendar data.

The project demonstrates practical SQL querying techniques used in Business Intelligence and Data Analysis workflows.

---
## Objectives
• Perform structured data analysis using SQL
• Explore customer behavior and sales trends
• Analyze currency exchange and transaction data
• Practice real-world Business Intelligence querying techniques
• Generate meaningful insights from relational databases
